#--------------------功能列表--------------------
1.添加客户：使用身份证号查询客户是否存在，不存在向数据库添加数据，同时加入缓存中。
2.修改客户：1.使用新身份证号查询客户（缓存和数据库）是否存在，
2.存在，不处理；3.不存在，3.1删除旧缓存，3.2修改数据库数据，3.3查询数据库（获得全部字段），加入缓存中。
